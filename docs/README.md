## 📦 快速参考

### 服务地址
- 后端API: http://localhost:3000
- 前端页面: http://localhost:8080

### API端点
- 健康检查: `GET /api/health`
- 机器人状态: `GET /api/robot/status`
- 控制指令: `POST /api/robot/control`
- 传感器数据: `GET /api/sensors`
- 任务列表: `GET /api/tasks`
- 作业统计: `GET /api/statistics`

### 测试文件
- Postman集合: `农业机器人API测试集合.postman_collection.json`
- Postman环境: `农业机器人-本地环境.postman_environment.json`
